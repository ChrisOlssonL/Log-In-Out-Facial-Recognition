## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

* Python 3
* pip
* OpenCV
* Some kind of camera

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ChrisOlssonL/Log-In-Out-Facial-Recognition.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your camera device in `VideoHandler\Video_Capture.py` (0 = Default) Change this number if no camera is recognised
   ```JS
   def __init__(self, v_source=0):
   ```
